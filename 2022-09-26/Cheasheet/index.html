Cheatsheat für Github

  1. Repository auf GitHub erstellen
  2. Code aus GitHub kopieren (das ist der Text der direkt kommt und man rauskopieren muss auf der Seite) und im Terminal einfügen und Github mit lokalem Repo verknüpfen
  3. Code aufschreiben (gemeint ist einfach an der index.html und style.css Datei arbeiten)
  4. git add *
  5. git commit -m "commit Nachricht"
  6. git push
